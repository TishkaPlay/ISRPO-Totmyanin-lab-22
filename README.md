# Калькулятор на JavaScript (Lab22)

## 📝 Описание проекта
Интерактивный веб-калькулятор, разработанный в рамках лабораторной работы №22. Приложение позволяет выполнять базовые арифметические операции, валидирует ввод пользователя и построено с использованием современных практик ES6 (модули, классы).

## ⚙️ Функциональность
- **Поддерживаемые операции:** сложение, вычитание, умножение, деление, десятичные дроби.
- **Ограничения:** запрещён ввод нескольких операторов подряд, нескольких точек в одном числе, выражений, начинающихся с оператора (кроме `-`).
- **Обработка ошибок:** при некорректном выражении или делении на ноль выводится сообщение `"Ошибка"`. Дальнейший ввод автоматически сбрасывает состояние.

### Класс `Calculator`
Инкапсулирует состояние (`expression`, `hasError`) и методы обработки. Отделён от DOM, что упрощает тестирование и повторное использование.

### Взаимодействие компонентов
`index.html` → загружает `main.js` (`type="module"`) → импортирует `calculator.js` → создаёт экземпляр `Calculator` → навешивает обработчики на кнопки → при клике вызывает методы класса → обновляет `#display`.

## 🔄 Логика работы
1. **Нажатие кнопки:** вызывается `handleButtonClick(value)`.
2. **Валидация:** `addToExpression()` проверяет корректность символа через `isOperator()`, `isLastCharOperator()`, `hasDuplicateDot()`.
3. **Вычисление:** `calculate()` проверяет `isValidExpression()`, затем вызывает `safeEvaluate()` (изолированный `eval` с фильтрацией символов).
4. **Отображение:** `updateDisplay()` рендерит строку или `0`/`Ошибка` в зависимости от состояния.
